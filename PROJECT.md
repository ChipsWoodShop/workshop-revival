# Project brief

## Purpose

Workshop Revival documents useful workshop projects built by repairing,
reusing, and repurposing materials that might otherwise be discarded.

**Brand:** Workshop Revival
**Tagline:** “Where old stuff gets a new life and earns its keep.”

## Technical scope

- Plain, semantic HTML
- One shared responsive CSS file
- Minimal, progressively enhanced vanilla JavaScript
- No framework, package manager, database, external CMS, paid service, or build
  dependency
- Relative internal URLs for compatibility with the project-site and custom
  domain URLs
- Static hosting on GitHub Pages with no additional hosting subscription

## Brand system

The working brand snapshot is in `docs/brand/`. It defines the approved logo,
web color tokens, and a helpful, hands-on, optimistic voice. The website uses
the supplied vector logo directly and does not introduce unapproved fonts or
competing color palettes. Obsidian remains the authoritative source; see
`docs/brand/SOURCE.md` for the vault-relative source notes and snapshot date.

## Hosting and domain

The production targets are `https://chipswoodshop.github.io/workshop-revival/`
and `https://workshoprevival.com/`. The domain is registered through
Squarespace, and DNS is managed there. Changes to GitHub Pages, Squarespace,
DNS, or other external accounts are outside repository work and require the
owner's explicit action.

## Content policy

Only verified project details, owner-provided photos, and confirmed publication
links should be published. Drafts and missing media must be visibly labeled.
Do not invent project claims, dates, materials, outcomes, or URLs.

## Project publishing

Project cards support both external publications and future builds hosted
directly on this website. An externally hosted card names its publisher and
links to the verified complete build; that presentation must not be baked into
the shared card structure as a requirement for every project.

The initial verified publications are:

- “Life... From Laundry! | DIY Grey Water Filter” on Instructables
- “Disc Brake Rotor Becomes a Monster Weed Puller” on Instructables

Workshop Revival is Chip's workshop in Long Beach, California. Its focus is
finding useful potential in would-be trash through practical repair,
repurposing, and revival.
