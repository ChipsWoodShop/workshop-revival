# Image assets

The approved Workshop Revival vector logo is stored in `brand/`.

Verified project thumbnails are stored in `projects/`:

- `grey-water-filter.jpg` — card thumbnail for “Life... From Laundry! | DIY
  Grey Water Filter”; 480 × 400 pixels
- `weed-puller.png` — card thumbnail for “Disc Brake Rotor Becomes a Monster
  Weed Puller”; 345 × 288 pixels

These small images are for project cards only. Do not upscale or reuse them as
hero, background, or full-width images. The similarly named `.webp` source
files supplied to the repository contain JPEG and PNG data respectively; the
correctly extended copies are the website-facing assets.

When real assets are added, use descriptive filenames, optimize them for the
web, and provide accurate alternative text in the relevant HTML.
